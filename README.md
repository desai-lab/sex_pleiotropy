This repo accompanies the research article "[**Sex decreases the pleiotropic costs of local adaptation by purging hitchhiking load**](https://doi.org/10.1101/2025.03.28.646016)" by Shreyas V. Pai†, Parris T. Humphrey†, Camille Simonet, Katya Kosheleva, Anurag Limdi, and Michael M. Desai* (2025).

It contains the variant calling pipelines and fitness assay analysis scripts used to generate all figures in the paper.

Raw FASTQ files are available at SRA accession number [PRJNA1379502](https://www.ncbi.nlm.nih.gov/sra/PRJNA1379502).
